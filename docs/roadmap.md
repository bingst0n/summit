# Summit — Roadmap

## Phase 1 — Foundation *(in progress)*
Get a working app you can actually use.

- [ ] Supabase tables created (goals, milestones, daily_logs)
- [ ] Fill `.env.local` with real keys
- [ ] Goals page: add/view/delete goals
- [ ] Auto-generate milestones distributed across the summer
- [ ] Check-in page: log daily rating + notes per goal
- [ ] Dashboard: streak, progress bars, today's status
- [ ] Nav component connecting all pages
- [ ] Deploy to Vercel + Pushcut cron wired up

## Phase 2 — Polish
Make it feel like a real app.

- [ ] PWA icons + install prompt
- [ ] Milestone completion toggling
- [ ] Goal detail view (full history, all milestones)
- [ ] Streak counter
- [ ] Basic stats (avg rating per goal, days logged)

## Phase 3 — Intelligence
Make it smarter.

- [ ] Weekly summary notification (what you hit, what you missed)
- [ ] Goal pacing warnings ("you're behind on X")
- [ ] Heatmap calendar of check-ins
